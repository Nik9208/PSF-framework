\begin{abstract}
A dynamical system does not necessarily possess a single fixed predictive representation. Instead, predictive structure depends on the prediction horizon, available information, representation space, and observer constraints. This paper introduces the Predictive Structure Framework (PSF), which formally defines an optimal predictive representation as the one that maximizes accessible predictive information under a temporal constraint: $\Phi^*(h) = \arg\max_\Phi I(\Phi,h)$. By treating machine learning models as measurement devices with inherent inductive biases, the framework separates physical predictive accessibility from observer-induced representation preference, offering an operational criterion for tracking how predictive structure appears, changes, and becomes inaccessible over time.
\end{abstract}
