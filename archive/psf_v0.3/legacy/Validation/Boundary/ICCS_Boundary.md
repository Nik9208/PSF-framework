# ICCS Boundary

## Overview
Current ICCS definition: Predictive Information + Stability + Observer Agreement.
We must find the absolute limits of this metric.

## Setup
Is there a system where:
- prediction is high
- stability is high
- observers agree

...but the extracted structure is merely an artifact?

## Failure condition
If such a system exists and yields a high ICCS, an additional constraint must be introduced to the framework.
