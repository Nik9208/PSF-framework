# Definitions

**Boundary:** A boundary is a controlled benchmark in which two or more systems are intentionally constructed to test whether a candidate metric can distinguish underlying structural differences under a specific ambiguity class.

**Requirements for a valid boundary:**
- Isolate a single ambiguity class.
- Minimize unrelated confounding factors.
- Admit reproducible generation.
- Produce quantitative pass/fail criteria.

**Orthogonal Descriptor:** A descriptor introduced to resolve a structural ambiguity that remains unresolved by the previously accepted descriptor set under the current boundary suite. Orthogonality refers to functional complementarity rather than linear algebraic orthogonality.

**Ambiguity Classes:**
- **Temporal Ambiguity:** Structurally different dynamical systems exhibiting equivalent short-term predictive measures.
- **Representation Ambiguity:** Equivalent dynamics producing different metric values under changes of representation or coordinate system.
- **Causal Ambiguity:** Equivalent predictive relationships arising from different causal structures.
- **Aggregation Ambiguity:** Loss of structural discrimination caused by collapsing multiple structural descriptors into a single scalar score.
