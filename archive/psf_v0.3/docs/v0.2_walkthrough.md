# PSF Framework v0.2: C-MAPSS Validation Suite

## Overview

This walkthrough summarizes the execution of the v0.2 validation suite for the Phase-Space Fingerprinting (PSF) framework. The validation extends the synthetic evaluation from v0.1 to multivariate sensor trajectories using the NASA C-MAPSS turbofan dataset.

The suite was designed to evaluate robustness against channel selection bias, representation changes, and observational noise.

## 1. Unbiased Channel Selection (Phase 0)

A fixed data-driven channel selection protocol was implemented:

- Variance filtering removed near-constant sensors.
- Spearman dependency clustering with Complete Linkage reduced redundant channels.

The resulting baseline representation:
X=s9, Y=s4, Z=s3

was selected without using RUL information or degradation labels.

## 2. Validation Matrix

### Temporal Boundary

ICCS components were evaluated across normalized Early, Middle, and Late degradation phases.

Finding:
The ICCS profile separates different response patterns. Memory-related and dependency-related measures change across degradation phases, while local geometric estimates remain comparatively stable.

### Representation Boundary

Multiple representations were tested:

- Raw
- Rolling features
- PCA-2
- PCA-1

Finding:
Representation transformations modify the observed ICCS profile. Smoothing can increase apparent temporal memory, while dimensionality reduction can remove parts of the multivariate dependency representation.

### Noise Boundary

Relative Gaussian noise was injected from 0% to 20%.

Finding:
Different ICCS components exhibit different sensitivity profiles. Local geometric estimates remain comparatively stable under tested noise conditions, while temporal and dependency estimates change gradually.
