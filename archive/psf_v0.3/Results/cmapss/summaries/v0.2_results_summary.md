# Experiment 1: Temporal Boundary

Question:
Does ICCS separate different structural changes during degradation?

Artifacts:
- cmapss_temporal_iccs.csv
- cmapss_temporal_boundary.md

Summary:
The temporal boundary experiment indicates that ICCS components exhibit non-uniform responses during degradation. Memory ($M(k)$) and dependency-related measures ($TE, CMI$) change substantially across phases, while local geometric estimates ($D_{local}$) remain comparatively stable. This suggests that scalar degradation signals combine multiple structural effects that can be separated by the ICCS representation.

---

# Experiment 2: Representation Boundary

Question:
Which representations preserve multivariate structure?

Representations:
- Raw
- Rolling
- PCA-2
- PCA-1

Summary:
Representation transformations are not neutral. Smoothing (`Rolling`) modifies temporal structure and can increase apparent memory and dependency estimates. `PCA-2` preserves aspects of the temporal signal while suppressing multivariate dependency structure through linear orthogonalization. `PCA-1` tests the loss of multivariate structure under one-dimensional projection, preserving scalar variance while removing cross-channel representation.

---

# Experiment 3: Noise Boundary

Question:
How do ICCS components degrade under observation noise?

Noise:
0-20% relative Gaussian noise

Summary:
The empirical noise boundary test provides evidence consistent with differential degradation under observational stress. Increasing noise attenuates memory ($M(k)$) and affects dependency estimates ($CMI$). However, the dependency signatures retain a substantial difference between Early and Late degradation phases up to 20% noise, while local geometric estimates ($D_{local}$) remain comparatively stable under the tested conditions.

---

# Overall Findings

Observed:
- components have different sensitivity profiles
- scalar measures combine multiple effects
- representation choice changes observed structure

# Limitations

- FD001 single operating condition
- TE/CMI interpreted as dependency estimates, not physical causality
