# Causal vs Predictive Divergence

## Overview
PSF claims accessible structure is determined by predictability.
However, Prediction $\neq$ Causation.

## Setup
System 1: $A \rightarrow B$
System 2: $C \rightarrow A, B$ (Common factor)
Both systems could have high $I(A_t; B_{t+1})$ but different causal graphs.

## Failure condition
If PSF assigns identical structure and claims it is "causally real", it fails the boundary test.
